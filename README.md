# IFC Inspect

Сервис создания смет строящегося дома по IFC и PDF документам.

## Структура проекта

```
ifc_inspect/
├── backend/
│   └── app.py              # Flask сервер (API)
├── frontend/
│   └── index.html          # Веб-интерфейс
├── scripts/
│   ├── page_splitter.py    # Разделение PDF на текстовые страницы и чертежи
│   └── ifc_parser.py       # Парсинг IFC файлов
├── data/                   # Справочники и данные
├── uploads/                # Загруженные файлы (создается автоматически)
└── requirements.txt        # Зависимости Python
```

## Возможности

1. **Загрузка файлов**: PDF (проектная документация) и IFC (информационная модель здания)
2. **Обработка PDF**: 
   - Разделение на текстовые страницы (≤ A4, ≤ 30см)
   - Выделение страниц с чертежами (> 30см)
   - Коррекция ориентации страниц
3. **Парсинг IFC**:
   - Извлечение информации о проекте
   - Подсчет элементов по типам (стены, перекрытия, колонны, балки и т.д.)
   - Извлечение материалов и их свойств
   - Генерация Excel отчета с подробной таблицей элементов

## Установка и запуск

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск сервера

```bash
cd backend
python app.py
```

Сервер запустится на порту 5001: http://localhost:5001

## API endpoints

- `GET /` - Веб-интерфейс
- `POST /api/upload` - Загрузка файлов
- `POST /api/process` - Обработка загруженных файлов
- `GET /api/results/<session_id>` - Получение результатов обработки
- `GET /api/download/<session_id>/<filename>` - Скачивание файлов (Excel отчеты)

## Алгоритм работы

1. Пользователь загружает PDF и IFC файлы через веб-интерфейс
2. Сервер сохраняет файлы в сессионную папку
3. При нажатии кнопки "Запустить анализ":
   - PDF файл разделяется на текстовые страницы и страницы с чертежами
   - IFC файл парсится, извлекаются данные об элементах здания
   - Генерируется Excel отчет с таблицей всех элементов
4. Результаты отображаются в веб-интерфейсе:
   - Количество текстовых страниц и чертежей
   - Количество элементов в IFC модели
   - Кнопка для скачивания Excel отчета

## Формат Excel отчета

Excel файл содержит:
- Лист "Сводка" - основная информация о проекте
- Листы по типам элементов (Стены, Перекрытия, Колонны, Балки, и т.д.)
- Для каждого элемента: ID, имя, тег, описание, материал, объем, площадь, размеры, свойства
